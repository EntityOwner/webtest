Создание закладки и использование её на chatgpt.com

Быстрый способ

    Создайте закладку быстрым способом: зажмите ЛКМ по кнопке ниже и перетащите её в панель закладок вашего браузера.
    Ссылка Stripe
    Если перетаскивание не работает — используйте ручной способ ниже.

Ручной способ

    Ручной способ: скопируйте код и создайте закладку вручную:
    Название — любое.
    URL — вставьте код из кнопки ниже

Использование на chatgpt.com

  1. Перейдите на chatgpt.com и выполните вход в свой аккаунт.
    Открыть ChatGPT
  2. Нажмите закладку в панели закладок (ниже адресной строки).
  3. Через 10–15 секунд появится окно со ссылкой вида https://checkout.stripe.com/... — скопируйте её.

Через консоль браузера (Stripe)

Получение ссылки через внутреннюю консоль браузера.

  1. Скопируйте код для вставки в консоль, нажав кнопку ниже.
    Открыть PlayCode
  2. Откройте chatgpt.com и выполните вход в свой аккаунт.
    Открыть ChatGPT
  3. Откройте консоль: нажмите F12
    (или Ctrl+Shift+I) ➡️ вкладка Console.
  4. Вставьте код в Console и нажмите Enter.
    В консоли появится ссылка
    https://checkout.stripe.com/...
    Скопируйте её и отправьте нам.
    Чат с нами на Plati.Market
    (последний заказ с Fursov)     Telegram менеджер

Если вместо Stripe‑ссылки появляется https://chatgpt.com/checkout/..., значит включён новый метод оплаты. В любом случае отправьте нам эту ссылку.

Копирование ссылки (по API)

Получение API‑ответа для дальнейшей покупки подписки менеджером.

  1. Откройте chatgpt.com и выполните вход в свой аккаунт.
  2. Нажмите кнопку ниже, для открытия chatgpt.com/api/auth/session.
    Открыть страницу API ответа
  3. Скопируйте весь длинный текст на странице:
    Нажмите Ctrl + A — выделить всё;
    затем Ctrl + C — скопировать.
  4. Вставьте весь скопированный текст в поле ниже и нажмите кнопку «Отправить менеджеру».
